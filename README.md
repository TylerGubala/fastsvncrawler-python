# fastsvncrawler-python
Python implementation of `fastsvncrawler` for efficient svn traversal in Python

There are three `git` submodules associated with this project:

[apr]()
[cpython]()
[subversion]()

## Usage

### Installation

#### Windows

`py -m pip install fastsvncrawler-python`

#### Linux

`python3 -m pip install fastsvncrawler-python`
